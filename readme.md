# Ringyo Zumen Maker

QGIS上のポイントレイヤから、林業向けの周囲図および排水・林内路網図を作成するプラグインです。地図画像、座標表、測量情報などをまとめたHTML図面を出力します。

## 仕様

北海道庁の[造林事業に係る造林実測図の作成方法](https://www.pref.hokkaido.lg.jp/fs/1/3/1/2/9/9/9/9/_/%E9%80%A0%E6%9E%97%E4%BA%8B%E6%A5%AD%E3%81%AB%E4%BF%82%E3%82%8B%E9%80%A0%E6%9E%97%E5%AE%9F%E6%B8%AC%E5%9B%B3%E3%81%AE%E4%BD%9C%E6%88%90%E6%96%B9%E6%B3%95.pdf)に基づいて作成しています。

## 動作環境

- QGIS 4.0以降
- Python 3.12以降（QGIS同梱環境）
- Pythonパッケージ `lxml`

動作確認環境: QGIS 4.0.1 Norrköping

## 主な機能

- 周囲ポイントから周囲線・面積・延長を計算
- 排水・林内路網を複数登録し、線形と延長を計算
- 除地を計算
- 測点名をQGIS式で指定
- QGIS式による対象ポイントの絞り込みと結合順の指定
- 指定した縮尺、CRS、磁補正を地図へ反映
- 測点座標表と測量情報を含むHTML図面を生成

## インストール

1. `RingyoZumenMaker.zip` をダウンロードします。
2. QGISを起動します。
3. 「プラグイン」タブの「プラグインの管理とインストール」を開きます。
4. 「ZIPからインストール」でダウンロードした`RingyoZumenMaker.zip` を読み込みます。

## 使用方法

### 1. 基本情報

「基本情報」タブで、測量日、測量者、林小班、山林所有者、製図情報などを入力します。図面の縮尺と、排水・林内路網の種別もここで指定します。

### 2. 座標系

「座標系」タブで出力図面のCRSと磁補正角度を指定します。磁補正角度は地図の回転角として使用されます。

### 3. 周囲図

周囲図を出力する場合は「周囲の図面を出力」を有効にし、次の項目を指定します。

- ポイントレイヤ
- 対象を絞り込むQGIS式（空欄の場合はレイヤ全体）
- 測点名のQGIS式
- ポイントの結合順を決めるQGIS式

### 4. 排水・林内路網図

排水・林内路網図を出力する場合は「排水・林内路網の図面を出力」を有効にします。「排水を追加」で入力欄を増やし、それぞれに名前、幅、ポイントレイヤ、測点名、結合順などを指定します。

### 5. 出力

「出力」タブで既存の出力フォルダを指定し、「実行」を押します。処理状況は下部のプログレスバーに表示されます。

## 出力ファイル

指定フォルダへ次のファイルを生成します。

```text
出力フォルダ/
├── index.html
└── asset/
    ├── index.css
    ├── houi.svg
    ├── shui_map.png
    ├── haisui_1_map.png
    ├── haisui_2_map.png
    └── mix_map.png
```

選択した製図種別や登録数によって、生成されるPNGファイルは異なります。`index.html` をWebブラウザで開くと図面を確認できます。
印刷する場合ブラウザの印刷機能を使用します。ページ上のセレクトボックス、ボタンなどは印刷時に非表示になります。

## 入力時の注意

- 入力にはポイントジオメトリのベクターレイヤを使用してください。
- 測点名の式は空欄にできません。
- CRSには有効な座標参照系を指定してください。
- 出力先には、あらかじめ存在するフォルダを指定してください。
- 同じ出力フォルダを使用すると、同名のHTML、画像、アセットが上書きされます。
- プラグインはプロジェクト内の `150x150_map` という印刷レイアウトを作り直します。
